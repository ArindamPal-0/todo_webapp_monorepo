> ## Setup for local development
> 
> Make sure to clone the repository first.
> ### **Windows**
> ```powershell
>   $ cd backend
>   $ python -m venv venv
>   $ venv/Scripts/Activate.ps1
>   $ pip install -r requirements.txt
>   $ python main.py
> ```
